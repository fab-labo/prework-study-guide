HTML

  • The !DOCTYPE html specifies what type of document the browser should expect when the page is opened
  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • The section elements are semantic elements that define a section in a document that contains thematic content
  • The ul tag creates an unordered list of items
  • The li element represents a new bullet point in a list
  • The head section contains information or metadata that helps the browser to render the page correctly.This code is not visible to the user
  • The div element creates a division in the page
  • The span elements creates an inline division in the page
  • The style element defines the CSS styles associated with a website
  • The base element defines the base URL for a webpage
  • The link element connects an external resource to the HTML document

CSS

  • The display property controls the box type that an element creates
  • The text color of an element is defined by its color parameter
  • The background properties are background-color, background-image, background-repeat, background-attachment, and background-position
  • The margin of an element is always translucent, independent of the backdrop color
  • Padding controls the distance between an elements content and border

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • Git clone downloads a copy of the project (remote repository) to your local computer
  • Git checkout selects the branch you want to work on
  • Git pulls recent changes from the remote repository to your local files
  • Git add stages file that you would like to commit

JavaScript

  • A variable is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script
  • The this keyword refers to a special property of an execution context
  • The function keyword defines a function expression
  • const is a keyword used to declare a variable that cannot be reassigned a new value
  • An array is a linear datat structure that groups elements that have the same data type
  • A string is an object that defines a series of characters
