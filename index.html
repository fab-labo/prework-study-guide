HTML

  • The !DOCTYPE html specifies what type of document the browser should expect when the page is opened
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The section elements are semantic elements that define a section in a document that contains thematic content.
  • The ul tag creates an unordered list of items
  • The li element represents a new bullet point in a list
  • The head section contains information or metadata that helps the browser to render the page correctly.This code is not visible to the user
  • The div element creates a division in the page
  • The span elements creates an inline division in the page
  • The style element defines the CSS styles associated with a website
  • The base element defines the base URL for a webpage
  • The link element connects an external resource to the HTML document.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
